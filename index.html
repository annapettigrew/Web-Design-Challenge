Summary Latitude Vs X

The purpose of this project was to analyze how weather changes as you move closer to the equator. In order to do this I pulled data from the OpenWeatherMap API to create a dataset of over 500 cities. After Creating the dataset, I used Matplotlib to chart different aspects of the weather vs. latitude. These factors included: Temperature, Cloudiness, Wind Speed, and Humidity. This website provides the Data and Visualizations that I created as part of this project.
